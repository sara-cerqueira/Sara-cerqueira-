# Sara-cerqueira-
Resumo do lab de fundamentos de cloud na DIO
🎯 O que aprendi
Durante o desenvolvimento do lab na DIO, aprendi os conceitos básicos de computação em nuvem e como eles são aplicados no Microsoft Azure.

 ☁️ Computação em Nuvem
Entendi que a computação em nuvem permite utilizar recursos como servidores, armazenamento e redes pela internet, sem precisar ter infraestrutura física.

 📌 Modelos de Nuvem
Aprendi que existem três tipos principais:
- Nuvem pública
- Nuvem privada
- Nuvem híbrida

 ⚙️ Modelos de Serviço
Também vi os modelos de serviço:
- IaaS: infraestrutura
- PaaS: plataforma
- SaaS: software

 💰 Custos
Compreendi a diferença entre:
- CapEx (investimento inicial)
- OpEx (pagamento conforme o uso)


 📈 Benefícios
A nuvem traz vantagens como:
- Escalabilidade
- Flexibilidade
- Redução de custos
  
 🚀 Conclusão
Esse lab me ajudou a entender melhor como funciona a computação em nuvem e a importância do Azure no mercado de tecnologia.
